# HoloInteract Device Management Dashboard

A React-based dashboard developed for the InnoXR Labs Software Development Intern Assessment.

## Features

* Secure Login Interface
* Device Management Dashboard
* Search Functionality
* Analytics Dashboard
* Device Interaction Bar Chart
* Device Status Pie Chart
* Campaign Performance Tracking
* Bonus Features Section

## Tech Stack

* React
* JavaScript
* CSS
* Recharts
* Vite

## Installation

```bash
npm install
npm run dev
```
