# HoloInteract Device Management Dashboard

## Overview

HoloInteract Device Management Dashboard is a React-based web application developed as part of the InnoXR Labs Software Development Internship Assessment.

The application allows administrators to monitor devices deployed across multiple customer locations, track device status, analyze performance metrics, and view campaign analytics through interactive dashboards.

---

## Features

### Authentication

* Login interface with validation
* Prevents login with empty credentials

### Device Dashboard

* Total Devices Overview
* Online Devices Count
* Offline Devices Count
* Device Monitoring Table
* Device Search Functionality

### Analytics Dashboard

* Device Interaction Analytics (Bar Chart)
* Device Status Distribution (Pie Chart)
* Campaign Performance Summary
* Performance Metrics Cards

### Bonus Features

* Remote Content Management
* Device Health Monitoring
* AI Visitor Insights
